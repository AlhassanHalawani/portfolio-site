VPN with DDNS

WireGuard server with Cloudflare API DDNS keeps your hostname reachable even when your IP changes.

  • WireGuard on Pi; peers for laptop/phone.
  • Python/Bash script updates Cloudflare A/AAAA record on IP change.
  • Optional: HAProxy/Nginx for split services behind VPN.
